외부 시스템 연동과 보안·운영을 고려한 백엔드 개발을 7년간 담당해 온 Java/Spring 개발자입니다. 10개 이상의 외부 API 연동, Java 17 마이그레이션 주도, 모의해킹 취약점 1,400건 개선 등 실제 운영 환경에서 검증된 경험을 바탕으로 신뢰할 수 있는 서비스를 만드는 데 집중해 왔습니다. 개발 시 정상 케이스뿐 아니라 예외 상황과 운영 중 발생할 수 있는 이슈까지 함께 고려하며, 특히 외부 연동이나 보안이 필요한 기능에서는 사소해 보이는 부분도 한 번 더 점검하는 편입니다. 이런 꼼꼼한 성향 덕분에 장애를 사전에 줄이고 안정적인 서비스 운영에 기여해 왔습니다.

나만의 강점

01 보안과 안정성을 최우선으로 생각하는 개발자

모의해킹 취약점 약 1,400건을 직접 분석하고 조치한 경험을 바탕으로, 보안을 후처리가 아닌 개발 단계부터 고려합니다. Spring Security, Oauth2, JWT, MFA, Jasypt 암호화 등을 전사 서비스에 적용하며 실제 운영 환경의 보안 수준을 높여왔습니다.

02 레거시를 현대화하는 마이그레이션 경험

Java 7 → 17 마이그레이션과 Spring Framework·ORM 업그레이드를 주도하며 오래된 시스템을 안정적으로 현대화한 경험이 있습니다. 세션 기반 인증을 Spring Security + JWT 방식으로 전환하는 등 레거시 코드를 점진적으로 개선하는 작업에 강점이 있습니다.

03 다양한 외부 시스템 연동 전문성

다우, 기프티콘, 컬쳐랜드 등 10개 이상의 외부 공급사 API 연동부터 SSO, KG통합인증, 검색엔진 연동까지 다양한 외부 시스템과의 통합 경험을 보유하고 있습니다. 각 연동마다 예외 처리와 운영 안정성을 함께 고려하여 장애 없이 서비스를 운영해왔습니다.
