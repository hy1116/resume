최현영

컴퓨터공학을 전공하고 올해로 7년 차를 맞이한 JAVA 백엔드 개발자 최현영입니다. 현재 비즈마켓 멤버십개발부 선임연구원으로 재직하며 모바일 쿠폰 시스템과 B2B 쇼핑 플랫폼의 핵심 로직을 담당하고 있습니다.

1. 견고한 백엔드 설계 및 인프라 경험

Spring Framework와 Spring Boot를 기반으로 10곳 이상의 신규 공급사 API 연동을 성공적으로 수행했으며, 보안 취약점 개선과 시스템 안정화에 주력해 왔습니다. 단순히 기능 구현에 그치지 않고 Nginx와 Linux 환경을 직접 구성하여 서비스 배포 환경을 최적화한 경험이 있습니다. 최근에는 Docker와 Kubernetes 같은 컨테이너 기술을 학습하며, 최신 인프라 트렌드에 빠르게 적응할 준비를 마쳤습니다.

2. 대량 트래픽 처리 및 성능 최적화 역량

특정 시간대 대규모 이벤트 트래픽이 집중되는 모바일 쿠폰 발송 서비스를 운영하며 성능 병목 현상을 해결해 왔습니다. 쿼리 튜닝을 통한 속도 개선뿐만 아니라, 프로세스 설계 자체의 문제점을 파악하고 비동기 배치 처리 등으로 로직을 분리하여 대량 트래픽 상황에서도 흔들리지 않는 대비책을 구축한 경험이 풍부합니다.

3. 협업과 성장을 지향하는 태도

다양한 프로젝트를 거치며 고객사의 요구사항을 기술적 해답으로 전환하는 능력을 길러왔습니다. 동료와의 원활한 소통을 바탕으로 복잡한 비즈니스 문제를 효율적으로 해결하는 데 보람을 느낍니다. 빠르게 변화하는 기술 환경 속에서 능동적으로 학습하고, 제가 가진 경험을 바탕으로 팀에 실질적인 기여를 하는 개발자가 되겠습니다.

4. 차별화된 강점: 다차원적 시각을 통한 정교한 예외 처리

설계 단계에서 단순 기능 구현을 넘어, 발생 가능한 예외 상황을 다각도로 고민하는 것이 저의 핵심 강점입니다. 남들이 무심코 지나칠 수 있는 로직의 허점을 미리 파악하고 방어 코드를 설계하여 장애 가능성을 사전에 차단하는 데 집중합니다. 이러한 꼼꼼함은 서비스의 연속성을 보장하며, 예측 불가능한 트래픽 상황에서도 시스템이 안정적으로 동작하게 만드는 원동력이 됩니다.

기술적 세부사항에 깊이 몰입하여 많은 시간을 투입하는 경향도 있었으나, 최근에는 AI 도구를 업무 프로세스에 적극 도입하여 이를 보완했습니다. 반복적인 검증 작업에 AI를 활용해 분석의 정밀함은 유지하면서도 작업 속도는 높였습니다. 이를 통해 확보한 리소스를 본질적인 설계와 고도화된 문제 해결에 투입하며, 개발 효율과 완성도를 균형 있게 관리하고 있습니다.

5. 가치관: 실행을 통해 얻는 데이터의 가치

'경험하지 못한 것에 대한 후회보다, 실행을 통해 얻는 데이터가 더 값지다'는 신념을 가지고 있습니다. 실패는 단순한 실수가 아니라 다음 의사결정을 정교하게 만드는 가장 확실한 지표라고 믿기 때문입니다.

이러한 가치관은 업무 태도로도 이어집니다. 설계나 로직에 의문이 생기면 이론에만 머물지 않고, 개인 프로젝트를 통해 직접 구현하며 가설을 검증합니다. 이 과정에서 오류를 해결하며 특정 기술의 장단점을 몸소 체득했고, 업계의 관행과 원칙들이 왜 생겨났는지 본질적으로 이해할 수 있었습니다. 직접 겪으며 얻은 인사이트를 바탕으로, 늘 근거 있는 설계를 제시하는 개발자가 되겠습니다.
