최현영

컴퓨터공학을 전공하고 올해로 7년 차를 맞이한 Java 백엔드 개발자 최현영입니다. 현재 비즈마켓 멤버십개발부 선임연구원으로 재직하며 모바일 쿠폰 시스템과 쿠폰사 연동 및 발송시스템 유지보수, 다양한 기업 고객의 요구사항에 맞춘 복지 포인트 몰 구축울 담당하고 있습니다.

1. 견고한 백엔드 설계 및 배포 환경 구성

Spring Framework와 Spring Boot를 기반으로 10곳 이상의 신규 공급사 API 연동을 수행했으며, 보안 취약점 개선과 시스템 안정화에 주력해 왔습니다. 기능 구현에 그치지 않고 Nginx와 Linux 환경을 직접 구성해 배포 환경까지 최적화했고, Jenkins로 빌드부터 배포까지 이어지는 CI/CD 파이프라인을 구축해 배포 과정을 자동화한 경험도 있습니다. 최근에는 Docker와 Kubernetes 같은 컨테이너 기술을 익히며 클라우드 네이티브 배포 환경으로 역량을 넓혀가고 있습니다.

2. 대량 트래픽 처리 및 성능 최적화 역량

매달 정해진 시간에 선착순 쿠폰 발행 이벤트를 진행하는 스타드림(KB국민카드) 서비스를 운영하며, 이벤트 시작 시점마다 사용자가 한꺼번에 몰려 발생하는 트래픽 폭주와 성능 병목을 여러 차례 해결해 왔습니다. 병목 구간을 분석해 인덱스를 추가하고 쿼리를 튜닝했으며, DB 커넥션 풀 사이즈를 조정해 동시 접속 처리 한계를 끌어올렸습니다. 당첨자 추첨처럼 즉시 처리하지 않아도 되는 로직은 배치로 분리해 트래픽을 시간축으로 분산시키고, 이벤트 시간대의 순간 부하에 대비해 서버 자원도 사전에 증설했습니다. 이렇게 병목을 해소한 뒤로는 약 3년간 이벤트를 안정적으로 운영했고, 그 과정에서 순간적으로 트래픽이 몰리는 상황을 어떻게 진단하고 대응해야 하는지 체득했습니다.

3. 협업과 성장을 지향하는 태도

KB국민카드, SC제일은행, 로레알몰, S-OIL 등 성격이 다른 10곳 이상 고객사 프로젝트를 진행하며, 저마다 다른 요구사항과 일정에 맞춰 이해관계자와 소통하고 조율하는 역량을 길러왔습니다. 이 과정에서 CMS 개선이나 대량발송 속도 개선처럼 지시받아 처리한 일이 아니라, 운영 중 발견한 문제를 팀장님과 운영팀에 직접 제안해 개선까지 이끌어낸 경험도 여러 차례 있습니다. 이렇게 협업 속에서 문제를 발견하고 해결해 나가는 과정에 보람을 느끼며, 최근에는 StockSense·APIverse 같은 개인 프로젝트로 FastAPI와 Kubernetes, AI 예측 모델처럼 실무에서 다뤄보지 못한 영역까지 직접 구현해보면서 역량을 계속 넓혀가고 있습니다.

4. 차별화된 강점: 다차원적 시각을 통한 정교한 예외 처리

설계 단계에서 단순 기능 구현을 넘어 발생 가능한 예외 상황을 다각도로 고민하는 것이 저의 핵심 강점입니다. 남들이 무심코 지나칠 수 있는 로직의 허점을 미리 짚어 방어 코드를 설계함으로써 장애 가능성을 사전에 차단하는 데 집중하며, 이런 꼼꼼함이 예측 불가능한 트래픽 상황에서도 시스템이 안정적으로 동작하게 만드는 힘이 됩니다.

다만 기술적 세부사항에 깊이 몰입하다 보니 시간이 오래 걸리는 경향도 있었는데, 최근에는 반복적인 검증 작업에 AI 도구를 적극 활용해 이를 보완하고 있습니다. 분석의 정밀함은 유지하면서 작업 속도를 높이고, 그렇게 확보한 시간을 본질적인 설계와 문제 해결에 투입해 개발 효율과 완성도를 함께 챙기고 있습니다.

5. 가치관: 실행을 통해 얻는 데이터의 가치

'경험하지 못한 것에 대한 후회보다, 실행을 통해 얻는 데이터가 더 값지다'는 신념을 가지고 있습니다. 실패는 단순한 실수가 아니라 다음 의사결정을 정교하게 만드는 가장 확실한 지표라고 믿기 때문입니다.

이 가치관은 업무 태도로도 이어집니다. 설계나 로직에 의문이 생기면 이론에만 머물지 않고 개인 프로젝트로 직접 구현하며 가설을 검증하고, 그 과정에서 마주친 문제를 풀면서 기술의 장단점과 업계 관행이 왜 자리 잡았는지를 체득해 왔습니다. 이렇게 직접 겪으며 얻은 인사이트를 바탕으로, 늘 근거 있는 설계를 제시하는 개발자가 되겠습니다.
