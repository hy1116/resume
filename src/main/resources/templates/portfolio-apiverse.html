APIverse

가입 즉시 샌드박스 키가 발급되는 Developer-First API 마켓플레이스 & 게이트웨이

사이트 바로가기 ↗ GitHub ↗

주요 기능

  • 1-Click 샌드박스: 가입 즉시 테스트용 API Key 자동 발급, 신용카드/승인 대기 없음
  • API 상세 페이지 내 Swagger UI 임베드 — 로그인 유저의 키가 자동 세팅되어 바로 호출 테스트
  • Curl/Java/JavaScript/Python/Go 코드 스니펫 즉시 복사 제공
  • WebSocket 기반 실시간 요청/응답 트래픽 모니터링
  • 무료 나눔 API → 등급별(Basic/Pro/Enterprise) 유료 구독 상품까지 아우르는 하이브리드 과금 모델
  • API 상품 등록 → 관리자 승인(approve) 플로우, 관리자 전용 어드민 콘솔 분리

아키텍처 & 모듈 구성

멀티 모듈 구조: core(공유 R2DBC 엔티티·레포지토리) / gateway(사용자 API, 8080) / admin(어드민 전용 API, 8090) / web·console(React SPA)로 관심사를 분리했습니다.

프록시 인증 분리: 일반 API는 JWT, /gateway/** 프록시 경로는 X-API-KEY 헤더로 별도 인증하며, 권한(role)과 과금 등급(tier)을 분리 관리해 등급 변경 API로 권한이 상승하는 취약점을 원천 차단했습니다.

완전 논블로킹: 모든 컨트롤러가 Mono/Flux를 반환하는 리액티브 스택으로 구성되어 있습니다.

보안 설계

  • API Key 무차별 대입 방어: 고엔트로피 키 발급 + IP 화이트리스트 필터로 유출 키 오용 차단
  • Redis Token Bucket 기반 Rate Limiting으로 DoS/DDoS 트래픽을 게이트웨이 진입 단계에서 컷오프
  • URL/쿼리 파라미터 화이트리스트 정규식 필터 + Named Parameter로 인젝션 공격 차단
  • 입점사 원본 인증 토큰은 AES-256 암호화 후 DB 저장, 복호화 키는 런타임 Secret으로 주입
  • Redis Lua Script로 쿼터 차감을 원자적 연산으로 묶어 동시 요청 레이스 컨디션 방지
