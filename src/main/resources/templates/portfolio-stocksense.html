StockSense

한국투자증권 Open API를 활용한 주식 포트폴리오 관리 및 AI 주가 예측 서비스

사이트 바로가기 ↗ GitHub ↗

주요 기능

  • 실시간 포트폴리오 조회 (잔고, 보유종목, 수익률)
  • 종목 상세 정보 및 차트 (OHLCV)
  • AI 기반 주가 예측 (XGBoost + LSTM 앙상블)
  • 주식 매수/매도 주문 (한국투자증권 API 연동, 모의투자 지원)
  • 거래량 상위 종목 조회 및 관심종목 · 가격 알림
  • Kafka 기반 실시간 가격 알림: Naver Finance 시세 폴링 → 조건 충족 시 알림 이벤트 발행

기술 스택

Backend
FastAPI SQLAlchemy 2.0 (async) PostgreSQL Redis Kafka 한국투자증권 Open API
ML / Data
XGBoost LSTM Argo Workflows (K8s CronJob) 뉴스 감성분석
Frontend
React Vite Chart.js Axios
Infra
Docker Kubernetes EFK (Elasticsearch/Fluentd/Kibana)

아키텍처 특징

ML 예측 파이프라인: 일별 시세 수집 → 전일 예측 평가 → 뉴스 크롤링/감성분석 → 피처 엔지니어링(MA5/10/20, RSI, Bollinger Bands, MACD) → XGBoost/LSTM 학습 순으로 이어지는 배치 파이프라인을 Argo Workflows로 자동화했습니다. 새 모델이 기존보다 성능이 좋을 때만 자동 활성화됩니다.

실시간 알림: 두 개의 백그라운드 태스크(price_producer / alert_consumer)가 Kafka 토픽을 통해 시세 폴링과 알림 조건 판정을 비동기로 분리 처리합니다.

운영 안정성: Redis 캐싱은 장애 시 자동 degrade 되도록 설계했고, 프로덕션 환경은 K8s 위에서 EFK 스택으로 로그를 수집합니다.
